### Task A — Verify Java installation
By the end of today you must have:
* javac working
* java working
* able to compile a .java file
* able to run a .java file
### Task B — Write your first Java program
Create a file named:
```
Day1.java
```
Program must print exactly:
```
Name: Madhan
Age: 18
```
(Replace age if needed.)
Compile:
```
javac Day1.java
```
Run:
```
java Day1
```
Expected output:
```
Name: Madhan
Age: 18
```
Once this appears, tasks are complete.
